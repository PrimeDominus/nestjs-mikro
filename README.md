# nestjs-mikro
Learning NestJS with Mikro-ORM
```

# start docker (linux)
``` command
sudo systemctl start docker
docker-compose up
```


# install package
``` command
npm install
npm run start:dev
npm run create:db
npm run start:dev
```
```
Now on your browser open this link to see the response 
http://localhost:3000/main
